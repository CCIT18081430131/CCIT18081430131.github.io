Docker

镜像操作命令

  • docker images # 查看镜像
  • docker rmi # 删除镜像
  • docker pull # 拉取镜像
  • docker push
  • docker save # docker save -o nginx.tar nginx:latest 保存镜像
  • docker load # docker load -i nginx.tar 读取镜像

容器相关命令

  • docker run

    docker run –name some-nginx -d -p 8080:80 some-content-nginx

    docker run –name doc-nginx -d -p 80:80 nginx
    docker run : 创建并运行一个容器
    –name : 给容器起一个名字，比如叫做mn
    -p : 将宿主机端口与容器端口映射，冒号左侧是宿主机端口，右侧是容器端口
    -d : 后代运行容器
  • docker pause # 暂停
  • docker unpause
  • docker stop
  • docker start
  • docker ps # 查看所有运行的容器及状态
  • docker logs # 查看容器运行日志
    docker logs doc-nginx -f # 实时查看
  • docker exec # 进入容器执行命令
    docker exec -it doc-nginx base
    -it : 给当前进入容器创建一个标准输入、输出终端，允许我们与容器交互
    base： 进入容器后执行的命令，base是一个linux终端交互命令
  • docker rm # 删除指定容器
