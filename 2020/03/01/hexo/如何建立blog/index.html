以下为我的安装全过程

本文参考文章

这里有几个我当时搭建是看的几个教程这里为大家奉上

推荐观看b站视频（我用的主题是yilia，里面有详细介绍安装和很多设置优化）

手把手教你从0开始搭建自己的个人博客 |无坑版视频教程| hexo

一步一步教你如何在Windows10上完成hexo博客搭建和yilia主题使用

一、安装Node.js

点击安装Node.js(安装时一直next就ok了)

打开cmd命令行输入node -v 和 npm -v，成功界面如下：

二、安装Git

点击安装Git(安装时一直next就ok了)

在cmd窗口输入git 若若结果列出Git的各种用法，则表示Git安装成功。

三、安装hexo

安装Hexo，在自己认为合适的地方创个文件夹（文件夹就是存放你的blog的文件），在文件夹中空白处鼠标右键打开git bash here

在git页面中输入npm install hexo -g，开始安装Hexo ，如下：

输入hexo -v，检查hexo是否安装成功 ，如下：

4、创建hexo

这样准备工作就完成了，接下来就开始正式创建hexo

在git（一定是从你刚刚建立的文件夹中打开的）中输入hexo init，初始化该文件夹（有点漫长的等待。。。）

看到后面的“Start blogging with Hexo！”，激动有木有！！！！！

输入npm install，安装所需要的组件

这时候，我们的本地hexo就已经建好了

输入hexo s，开启服务器，就可以本地访问我们的网站了。

在游览器中打开 http://localhost:4000/ 就可以本地访问了
