hexo博客搭建教程（2）：hexo部署到github教程
Created 2020-03-05| Updated 2022-06-18|hexo
Post View:

1、注册，新建仓库

进入官网 点击右上角sign up 注册。（过程可能会比较慢，耐心等待）

注册好了之后点击 create respository 按钮 新建仓库。（仓库名必须是你刚刚注册的使用的用户名加.github.io ，如：我的github用户名是xiaotaiyang，那我的仓库名必须设置为xiaotaiyang.github.io ） 填好仓库名就可以直接创建了，其他的选项可以不填。
